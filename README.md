# Coffee-Shop-Saless
Coffee Shop Sales Dashboard

This project is an interactive dashboard built to analyze coffee shop sales data and understand customer behavior.

📊 Features
Total Sales, Footfall & Average Bill analysis
Hourly order trends
Category-wise sales distribution
Store-wise performance comparison
Top selling products
Weekday vs weekend order analysis
🛠️ Tools Used
Microsoft Excel (Data Cleaning & Dashboard)
🚀 Purpose

To analyze sales data, identify trends, and improve decision-making using data visualization.
